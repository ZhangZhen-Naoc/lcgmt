You can cross-match by uploading a catalog with specific format:

If you upladed a csv file, please make sure it:
  1. Is sperated by ","
  2. At least has 4 column: id,ra,dec,pos_err
  3. Unit: ra in degree, dec in degree, pos_err in arcsec
  4. Has header
Download csv Example
If you upladed a fits file, please make sure it:
  1. At least has 2 column: ra,dec.
  2. If it has different column name, you can customize the column name
  3. If it doesn't has pos_err column, you can customize the value. But pay attention: This will overwrite the pos_err_col
  4. Unit: ra in degree, dec in degree, pos_err in arcsec
