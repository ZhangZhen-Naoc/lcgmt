Notes:

  1. The input PSF file is based on the calibration of QM4, while the input ARF and RMF are simulated by Geant4 (Zhao et al 2017, Experimental Astronomy, 43, 267).
  2. For the X-ray background in 0.1-2 keV, the data observed with ROSAT has been used (Snowden et al 1997, ApJ, 485, 125). The background in 2-10 keV and the particles background are generated by Geant4 (Zhao et al 2017, Experimental Astronomy, 43, 267).
  3. The output files include events files (event.fits), images (image.fits), light curves (in 0.5-4 keV, lc.fits), source spectra (spec.fits), background spectra (bkgspec.fits), response and ancillary files (rmf.fits, arf.fits). For sources detected in the center, midway, and corner of the FoV, the output data are labeled by 1, 2, 3, respectively, except that the same rmf file can be used for all the spectra. The data are compatible with common software such as ds9, Xspec and Xronos.
